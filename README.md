# Break-even Lab

An interactive web application designed to support learners on the Level 1/2 First Business course, specifically for Unit 2: Finance in Business.

## Purpose

This tool helps students understand break-even point (BEP) concepts through:

- Interactive break-even chart creation
- Visual representation of total cost, total revenue, and fixed cost lines
- Calculation of break-even point and margin of safety
- What-if scenario analysis with sliders
- Interactive exercises to identify profit and loss regions
- Card sorting activity for advantages and disadvantages of break-even analysis

## Features

- **Chart Building**: Enter price, variable costs, and fixed costs to generate break-even charts
- **Interactive Learning**: Click-to-identify exercises for chart elements
- **Formula Practice**: Calculate BEP and margin of safety with step-by-step explanations
- **Scenario Testing**: Use sliders to see how changes affect the break-even point
- **Assessment Activities**: Sort advantages/disadvantages cards to test understanding

## Usage

Open `index.html` in a web browser. No internet connection required - all functionality works offline for classroom use.
